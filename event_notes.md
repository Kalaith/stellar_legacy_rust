# Long-Term Expedition Events

For **Stellar Legacy**, I'd also include events that only make sense on voyages lasting centuries:

* A crew awakens to discover their home civilization has vanished.
* The ship's mission becomes obsolete after political changes back home.
* Successive generations reinterpret the mission into a religion.
* The ship splits into factions with competing goals.
* Multiple descendant cultures emerge aboard the same vessel.
* The onboard AI becomes a historical authority, preserving centuries of memory or subtly rewriting it.
* New species evolve inside forgotten maintenance shafts.
* A drifting fleet of other century ships forms an impromptu civilization.
* Long-range probes reveal that stars themselves are changing in unexpected ways.
* Crew descendants choose to settle a world, leaving the original mission understaffed.

---

For a game like **Stellar Legacy**, I'd recommend organizing encounters into reusable systems rather than one-off stories. Think in terms of event "families" such as:

| Category    | Examples                                                            |
| ----------- | ------------------------------------------------------------------- |
| Exploration | First contact, ruins, anomalies, strange worlds                     |
| Diplomacy   | Trade, war, refugees, negotiations, cultural misunderstandings      |
| Engineering | Reactor failures, hull damage, AI faults, shortages                 |
| Biology     | Disease, mutations, alien ecosystems, parasites                     |
| Science     | New particles, spatial phenomena, temporal effects                  |
| Survival    | Scarcity, morale crises, environmental hazards                      |
| Mystery     | Ghost signals, abandoned ships, impossible artifacts                |
| Comedy      | Bureaucrats, pranksters, translation mishaps, harmless infestations |
| Ethics      | Rescue decisions, intervention, conflicting obligations             |
| Legacy      | Generational change, mission drift, forgotten history               |

This structure lets you generate thousands of distinct missions by combining categories, complications, and outcomes. For example: *"A refugee convoy arrives during a reactor coolant leak while a nearby temporal anomaly causes messages from the ship's future to appear."* None of those elements are unique on their own, but together they create a memorable scenario with meaningful choices.
